# ESG 데이터 — 데이터 소스 정리

플랫폼 **도표 및 그림 생성** (`/charts`) 페이지의 **데이터 소스** (차트용 + ESG DATA 테이블용)만 E/S/G 탭별로 정리합니다.

---

## 1. Environmental (환경)

### 1-1. 차트용 데이터 소스

| 카테고리 | 데이터 소스 |
|----------|-------------|
| **온실가스/에너지** | Scope 1+Scope 2 총 배출량, Scope 1 총 직접 배출량, Scope 2 총 간접 배출량, Scope 3 총 기타간접 배출량, 총 에너지 소비량, 재생에너지 사용 현황, 탄소 배출량 데이터, 에너지 사용량 데이터 |
| **폐기물/대기** | 총 폐기물 발생량, 화학물질 배출량, 폐기물 현황(처리 유형별), 폐기물 현황(종류별), NOx 배출량, SOx 배출량, 먼지 배출량(TSP), VOC 배출량, 유해화학물질 배출량 |
| **용수/폐수** | 용수 사용량, 폐수 처리량, 용수 취수량, 용수 재활용량, TOC 배출량, BOD 배출량, TN 배출량 |

### 1-2. ESG DATA 테이블 — 데이터 소스

| 프리셋 | 개별 테이블 목록 |
|--------|------------------|
| **온실가스 배출량** | Scope 1, 2 온실가스 배출량, Scope 3 온실가스 배출량 |
| **에너지/재생에너지** | 에너지사용량, 본사 데이터센터 재생에너지 사용량 |
| **투자/PUE** | 친환경 투자내역, 데이터센터 평균 PUE, 데이터센터별 PUE |
| **용수/폐수** | 용수 취수량 및 방류량, 용수 사용량 |
| **폐기물/대기** | 폐기물 발생량, 본사 폐기물 유형별 처리량, 환경·에너지경영시스템(ISO 14001 & ISO 50001), 물리적 기후 리스크 |

## 2. Social (사회)

### 2-1. 차트용 데이터 소스

| 카테고리 | 데이터 소스 |
|----------|-------------|
| **임직원/교육** | 임직원 수(2024), 장애인 고용 현황, 국내 임직원 현황, 해외 임직원 현황, 여성 임직원 현황(국내), 여성 임직원 현황(해외), 퇴직 현황, 교육훈련 현황, 인적자본 ROI |
| **안전보건/윤리** | 성과 평가, 조직 건강도, 산업재해율, 근로손실재해율, 총 기록재해율, Fatality 및 직업성질환, 산업안전 교육, 윤리경영 제보·상담·발견, 윤리경영 위반 징계 현황 |
| **사회공헌/협력사** | 사회공헌 활동 비용, ESG리스크 평가 협력사 수, 봉사활동 시간, 협력사 구매액, 개선 계획 수립 협력사 수 |

### 2-2. ESG DATA 테이블 — 데이터 소스

| 프리셋 | 개별 테이블 목록 |
|--------|------------------|
| **인력/구성** | 글로벌 인력 현황, 국적별 인력 현황 |
| **채용/교육/역량** | 채용 현황, 임직원 교육 훈련 현황, 핵심 역량 개발 프로그램 참여율, 교육 프로그램별 현황, 인적자본 ROI |
| **다양성/근속/육아** | 여성 임직원 비율, 근속 기간, 퇴직 현황, 육아휴직 현황 |
| **안전/보건/보상** | 성과 평가, 조직 건강도, 노사협의회 운영 현황, 임직원 보수, 산업안전보건, ISO 45001 적용 범위 |
| **협력회사/공급망** | 협력회사 현황, 협력회사 지원 규모, 공급망 ESG 관리 현황, 협력회사 현장 점검 주요 항목별 준수율 |
| **고객/채널/개인정보** | 고객 지원, 장애 발생 건수 및 고객 다운타임, 유통 채널 관리 및 실사, 개인정보 관리 현황 |

## 3. Governance (거버넌스)

### 3-1. 차트용 데이터 소스

| 영역 | 데이터 소스 |
|------|-------------|
| **이사회** | 이사회 개최 수, 안건 수, 출석률, 위원회 개최 수(위원회별), 여성 이사 수·비율, 이사 보수 합계, 이사회 주식 보유 현황 |
| **컴플라이언스/부패/정보보안** | 총 부정부패 발생 건수, 유형별 부정부패 제보(합계·건수·비율), 위반 사건(부패 및 뇌물수수), 법적 위험 평가 사업장 수, 법적 제재 현황, 정보보안 사고/위반·데이터 누출·벌금/과태료 |

### 3-2. ESG DATA 테이블 — 데이터 소스

| 프리셋 | 개별 테이블 목록 |
|--------|------------------|
| **이사회/보수/위원회** | 이사회 현황, 이사회 운영 성과, 이사회 내 위원회 개최 횟수, 이사의 보수, 이사회 주식 보유 현황 |
| **컴플라이언스/부패/정보보안** | 부정부패 발생 건수, 유형별 부정부패 제보 현황, 위반 사건 발생 현황, 부패 관련 법적 위험 평가, 법적 제재 현황, 정보보안 관리 현황 |
