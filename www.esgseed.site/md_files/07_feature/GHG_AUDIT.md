# GHG 산정 — 공시 프레임워크별 데이터셋 및 증빙·리포트

본 문서는 GHG 산정 페이지에서 **공시 프레임워크 선택** 시 **산정**과 **감사·검증 대응**에 필요한 데이터셋을 정리하고, **증빙 패키지 내보내기**·**리포트 생성**에서 다운로드할 수 있는 데이터 및 포함 내용(컬럼명 등)을 상세히 기술합니다.

---

## 1. 공시 프레임워크별 산정·감사대응 데이터셋

### 1-1. 프레임워크별 적용 범위

| 공시 프레임워크 | 산정 입력 탭 | Scope | 감사 체크리스트 |
|-----------------|--------------|-------|-----------------|
| **IFRS S2** (ISSB) | 전력·열·스팀, 연료·차량, 냉매, 폐기물, 물류·출장·통근, 원료·제품 (6탭) | 1·2·3 | ISSB_CHECKLIST (12항목) |
| **GRI 305** | 동일 (6탭) | 1·2·3 | ISSB_CHECKLIST |
| **KSSB** (국내 기후 공시) | 동일 (6탭) | 1·2·3 | ISSB_CHECKLIST |
| **K-ETS** (배출권거래제) | 전력·열·스팀, 연료·차량, 냉매 (3탭). 폐기물·물류·원료 탭 숨김 | 1·2 | KETS_CHECKLIST (10항목) |
| **ESRS E1** | 동일 (6탭). Scope 2 location/market 기반 필수 | 1·2·3 (15 cat) | ISSB_CHECKLIST |

### 1-2. 산정용 필요 데이터셋 (탭별)

| 탭 | 필수 데이터 | 컬럼(필드) 예시 |
|----|-------------|-----------------|
| **전력·열·스팀** | EMS 에너지 사용량 | 월, 사업장, 에너지원, 사용량, 단위, 출처(ERP/엑셀/수동) |
| **연료·차량** | ERP 연료 구매·차량 연료 | 월, 사업장, 연료, 사용량, 단위, 배출량(tCO₂e) |
| **냉매** | EHS 냉매 충전·탈출 | 월, 사업장, 냉매 종류, 사용량, 단위 |
| **폐기물** | EMS 폐기물 발생·처리 | 소각(Scope 1), 위탁·매립·재활용(Scope 3) |
| **물류·출장·통근** | SRM·HR 물류·출장·통근 | 운송 유형, 거리, 연료, 배출량 |
| **원료·제품** | SRM·PLM 원료·제품 탄소 | 카테고리, 공급량, 단위 |

### 1-4. 프레임워크별 리포트 안내

| 공시 프레임워크 | 리포트 특징 |
|-----------------|-------------|
| **K-ETS** | 월별 에너지 사용량·Scope 1 중심. 공시_월별에너지사용량 시트 필수 |
| **IFRS S2** | 연간 총합, Scope 2 위치/시장 기반, Scope 3 중대 카테고리 |
| **KSSB** | 국내 ISSB 기반, Scope 3 유예/중대성 판단 |
| **GRI 305** | GHG Protocol 기반 Scope 1·2·3 |
| **ESRS E1** | double materiality, 15개 Scope 3 카테고리 |

## 2. 증빙 패키지 내보내기 — 다운로드 항목

감사·검증 대응 탭의 **증빙 패키지 내보내기**에서 다운로드할 수 있는 패키지 항목입니다. 프레임워크 선택에 따라 파일명·체크리스트 내용이 자동 변경됩니다.

| 파일명(예시) | 포함 내용 | 대상 |
|--------------|-----------|------|
| **GHG_산정결과_2024.pdf** | Scope 1/2/3 배출량·산식·배출계수 요약 | 공시 제출 |
| **GHG_데이터계보_2024.xlsx** | 전체 레코드 `source_system`, `synced_at`, 활동자료→배출계수→결과 연결 | 감사인 |
| **GHG_수동조정이력_2024.xlsx** | 수정 건 `전후값`, `사유`, `수정자`, `수정일시` | 감사인 |
| **GHG_배출계수적용내역_2024.xlsx** | `applied_factor_id`, `applied_factor_value`, 적용 시점 스냅샷 | 감사인 |
| **GHG_IFRS_S2_체크리스트_2024.pdf** 또는 **GHG_K-ETS_체크리스트_2024.pdf** | 12개(IFRS S2) 또는 10개(K-ETS) 요건 충족 여부·근거 요약 | 검증의견서 |

- K-ETS 선택 시: 체크리스트 파일명 `GHG_K-ETS_체크리스트_YYYY.pdf`
- IFRS S2 / GRI / KSSB / ESRS: `GHG_IFRS_S2_체크리스트_YYYY.pdf`

### 3-3. 검증의견서 부속 서류 6항목 (PDF·Excel 일괄 다운로드 포함)

| 항목 | 포함 내용 |
|------|-----------|
| **1. 검증 범위 요약** | 조직 경계(연결 기준), 물리적 경계(사업장 목록), 운영 경계(Scope 1/2/3 포함 여부), 제외 항목 및 사유 |
| **2. 배출량 집계표** | [사업장명] \| [Scope 1] \| [Scope 2] \| [Scope 3] \| [합계]. 가스별(CO₂, CH₄, N₂O, HFCs, PFCs, SF₆) tCO₂e. 사업장별 기여도 파이 차트 |
| **3. 데이터 품질 통계** | 실측 vs 추정 비율, DQR 분포, EMS/ERP 자동 적재 vs 수기 입력 비율 |
| **4. 적용 배출계수·산식 요약** | 에너지원별 배출계수, 단위, 출처(IPCC, 환경부 등). 산식: Activity × EF × GWP = tCO₂e. 계수 변경 이력 |
| **5. 활동자료 요약** | 에너지원별 총합(전력 MWh, LNG Nm³ 등), 월별 추이, 이상치 검증 |
| **6. 증빙 목록** | [전표번호/파일명] \| [연결된 활동자료 행] \| [업로드 일시]. 미매핑 주요 활동자료 경고 |

### 3-4. JSON 다운로드

| 최상위 필드 | 내용 |
|-------------|------|
| `generatedAt` | 생성 일시 (ISO 8601) |
| `boundaryPolicy` | 조직 경계, 보고 연도, EF DB 버전 등 |
| `scope1` | stationary, mobile 배열 |
| `scope2` | electricity 배열 |
| `scope3` | categories 배열 (category, data, receipts) |
| `disclosureFramework` | 공시 기준 (ISSB, K-ETS 등) |
| `summary` | totalTco2e, scope1Tco2e, scope2Tco2e, scope3Tco2e |

각 활동자료 행 필드 예: `id`, `year`, `month`, `facility`, `energySource`, `amount`, `unit`, `emissions`, `dataQuality`
