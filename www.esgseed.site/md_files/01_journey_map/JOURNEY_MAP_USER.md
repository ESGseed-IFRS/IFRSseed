# IFRSseed AI 플랫폼 — 사용자별 종합 저니맵

이 문서는 **시스템 전체**에서의 사용자 여정을 **사용자 유형별**로 종합 정리한 저니맵입니다. 로그인·회원가입부터 회사정보·GHG 산정·SR 작성·도표 생성·대시보드·보고서 출력까지 **한 번에 보는 총괄 플로우**를 담았으며, 각 페이지의 상세 단계는 하단 참조 문서를 따릅니다.

## 3. 종합 저니맵 — ESG 팀장(결재권자)

### 3.1 전체 플로우 개요

```
[1] 로그인·계정
    └ 회사 등록(최초) → DART 회사정보 자동 불러오기 → 초대 링크 발송
       → 초대 링크에서 ESG팀장(관리자) 등록 → 비밀번호 설정 → 관리자 대시보드
    └ 이후 이용 시: 이메일/비밀번호 또는 소셜 로그인 → 대시보드/권한별 메뉴

[2] 대시보드 (허브)
    └ 전체 진행률·마감일·D-day 확인
    └ 페이지별 진행 상태(회사정보·GHG·SR·도표) 카드·담당자·이동 →
    └ 확인 필요 사항·다음 액션 → 해당 페이지로 이동
    └ 팀원 현황: 가입 승인 대기(승인/반려), 팀원 목록, + 팀원 초대
    └ 보고서 완성도 스코어카드(회사정보 제출·GHG 저장·SR 90%·차트 저장) → 모두 충족 시 최종 승인
    └ 최근 활동 로그
    └ 최종 승인 후 Excel·PowerPoint 다운로드

[3] 회사정보
    └ 기업 기본정보·연락처·ESG 목표·이해관계자 (DART 자동 + 수기 입력)
    └ 시스템 연동 카드 → 적재된 데이터 조회
    └ 정보 확인 및 검증 → 최종 보고서에 제출

[4] GHG 산정
    └ GHG 산정·공시 탭: 공시 프레임워크 선택·6탭(전력·연료·냉매·폐기물·물류·원료)
    └ 전력/연료: STEP 1 데이터 확인 → STEP 2 산정 실행 → STEP 3 결과 확인 → 이 탭 결과 저장
    └ EMS 불러오기·엑셀 업로드·전체 결과 저장·CSV 내보내기·계수 기준 연도
    └ 감사·검증 대응 탭: 요건 체크리스트·데이터 계보·증빙 패키지 내보내기 등
    └ 리포트 생성 탭: PDF Report·Excel Details·일괄 다운로드

[5] SR 작성
    └ Table of Contents에서 항목 선택 → 직접 작성 또는 AI 문단 생성 → 저장
    └ 전체 준수율·관련 공시 기준 확인 → 시각화 추천·도표 생성으로 이동
    └ 저장 후 대시보드에서 진행 현황 확인

[6] 도표 및 그림 생성
    └ Environmental/Social/Governance 탭 → 데이터 소스·차트 유형·차트 생성하기 → 저장
    └ ESG DATA 테이블 입력·저장 → 도표 갤러리·최종보고서 반영
    └ 대시보드에서 확인

[7] 보고서 출력 (대시보드)
    └ 스코어카드 4항목 충족 → 최종 승인 → Excel(.xlsx)·PowerPoint(.pptx) 다운로드
```

### 3.2 단계별 요약 표 (ESG 팀장)

| 단계 | 구간 | 사용자 행동 요약 | 주로 보는 화면 |
|------|------|------------------|----------------|
| 1 | **계정** | 회사 등록(최초) 또는 로그인 | 로그인/회원가입·회사 등록·초대 링크 페이지 |
| 2 | **허브** | 대시보드에서 진행률·마감일·팀원 현황·확인 필요 사항·스코어카드·최종 승인·다운로드 | 대시보드 (전체 진행률·페이지별 카드·팀원 현황·보고서 출력·활동 로그) |
| 3 | **회사정보** | DART+수기 입력·검증·제출 | 회사정보 관리 (4개 섹션·보고서 요약 미리보기·시스템 연동) |
| 4 | **GHG** | 6탭 산정 입력·EMS/엑셀·스텝 플로우·결과 저장·감사 탭·리포트 생성 | GHG 산정·공시 / 감사·검증 대응 / 리포트 생성 |
| 5 | **SR** | 목차 선택·직접 작성/AI 문단 생성·저장·준수율 확인 | SR 보고서 작성 (목차·에디터·준수율·시각화 추천) |
| 6 | **도표** | ESG 탭·데이터 소스·차트 생성·저장·테이블 저장 | 도표 및 그림 생성 (Environmental/Social/Governance·차트 갤러리·도표 갤러리) |
| 7 | **출력** | 최종 승인·Excel/PPT 다운로드 | 대시보드 (보고서 출력 블록) |

## 4. 종합 저니맵 — ESG 팀원

### 4.1 전체 플로우 개요

```
[1] 로그인·계정
    └ 회원가입 → 회사 정보 확인(미등록 시 "회사 등록 먼저 필요") → 이메일 인증 → 비밀번호 설정
       → HRIS 조회로 역할 자동 할당 또는 승인 대기 → ESG 팀장 승인 후 로그인 가능
    └ 이후 이용 시: 이메일/비밀번호 또는 소셜 로그인 → 권한별 메뉴

[2] 대시보드 (허브)
    └ 전체 진행률·페이지별 진행 상태 확인 (담당자 표시 없음)
    └ 확인 필요 사항·다음 액션 → 본인 담당 항목 클릭하여 해당 페이지 이동
    └ 팀장 피드백: 수정 요청·승인 완료 확인 → 수정 후 재요청
    └ 보고서 완성도 스코어카드·섹션별 진행 (최종 승인 버튼 비표시)
    └ 팀장 최종 승인 후 Excel·PowerPoint 다운로드

[3] 회사정보
    └ 기업 기본정보·연락처·ESG 목표·이해관계자 확인 (조회 중심 또는 담당 섹션만 수정)
    └ 시스템 연동 → 적재된 데이터 조회. 수정 필요 시 팀장에게 요청

[4] GHG 산정
    └ 담당 탭(예: 전력·열·스팀) 선택 → STEP 1~3 진행
    └ 권한 시 EMS 불러오기·엑셀 업로드·이 탭 결과 저장
    └ 감사·검증 대응 탭: 요건 체크리스트·데이터 계보 조회 (데이터 마감·감사인 뷰는 팀장 전용)
    └ 리포트 다운로드는 권한 시 또는 팀장 요청

[5] SR 작성
    └ 목차 선택·직접 작성 또는 AI 문단 생성·저장
    └ 준수율·시각화 추천 확인 → 도표 생성으로 이동(필요 시)
    └ 저장 후 대시보드에서 진행 현황 확인

[6] 도표 및 그림 생성
    └ ESG 탭·데이터 소스·차트 생성·저장·테이블 저장 (팀장과 동일 화면, 권한 범위 내)

[7] 보고서 출력 (대시보드)
    └ 팀장 최종 승인 전: 스코어카드·섹션별 진행만 확인
    └ 팀장 최종 승인 후: Excel·PowerPoint 다운로드
```

### 4.2 단계별 요약 표 (ESG 팀원)

| 단계 | 구간 | 사용자 행동 요약 | 주로 보는 화면 |
|------|------|------------------|----------------|
| 1 | **계정** | 회원가입·회사 확인·인증·비밀번호·승인 대기(해당 시) 또는 로그인 | 회원가입·로그인 페이지 |
| 2 | **허브** | 대시보드에서 진행률·확인 필요 사항·팀장 피드백·스코어카드·승인 후 다운로드 | 대시보드 (진행률·페이지별 카드·팀장 피드백·보고서 출력) |
| 3 | **회사정보** | 조회·권한 시 일부 수정·수정 요청(팀장) | 회사정보 관리 (읽기 전용 또는 담당 섹션) |
| 4 | **GHG** | 담당 탭 입력·스텝 플로우·권한 시 저장·감사 탭 조회 | GHG 산정·공시 (담당 탭)·감사·검증 대응 (조회) |
| 5 | **SR** | 목차 선택·직접 작성/AI 문단·저장·준수율 확인 | SR 보고서 작성 |
| 6 | **도표** | 차트·테이블 생성·저장 | 도표 및 그림 생성 |
| 7 | **출력** | 팀장 승인 후 Excel/PPT 다운로드 | 대시보드 (보고서 출력 블록) |

## 6. 권장 이용 순서 (팀장 기준)

보고서 완성도를 채우기 위한 **논리적 순서**는 다음과 같습니다. 실제로는 대시보드에서 확인 필요 사항을 따라 필요한 페이지만 방문해도 됩니다.

| 순서 | 페이지 | 목적 |
|------|--------|------|
| 1 | **회사정보** | 기업 기본정보·연락처·ESG 목표·이해관계자 입력·검증·제출 → 스코어카드 "회사정보 제출" ✓ |
| 2 | **GHG 산정** | 6탭(전력·연료 등) 데이터 입력·산정·전체 결과 저장 → 스코어카드 "GHG 산정 결과 저장" ✓ |
| 3 | **SR 작성** | 목차별 본문 작성(직접/AI)·저장 → 스코어카드 "SR 준수율 90% 이상" 충족 |
| 4 | **도표 및 그림 생성** | 차트·ESG DATA 테이블 저장 → 스코어카드 "차트·도표 저장" ✓ |
| 5 | **대시보드** | 스코어카드 4항목 확인 → 최종 승인 → Excel·PowerPoint 다운로드 |

## 7. 요약

- **문서 성격**: 시스템 **전체**의 사용자 여정을 **ESG 팀장**·**ESG 팀원** 두 유형별로 종합한 저니맵.
- **플로우**: 로그인/회원가입(경로 분리) → **대시보드(허브)** ↔ 회사정보 ↔ GHG 산정 ↔ SR 작성 ↔ 도표 및 그림 생성 → 대시보드에서 **최종 승인·보고서 출력(Excel/PPT)**.
- **팀장 전용**: 회사 등록·마감일·팀원 현황(승인/반려·초대)·최종 승인·최근 활동 로그.
- **팀원 전용**: 팀장 피드백 확인·재요청·승인 후 다운로드. 회사정보/GHG는 조회 또는 담당 범위만 수정.
- **상세 단계**: 로그인·회사정보·GHG·SR·도표·대시보드 각각의 화면별 단계·6열 표는 **JOURNEYMAP_LOGIN.md**, **JOURNEYMAP_COMPANY_INFO.md**, **JOURNEYMAP_GHG.md**, **JOURNEYMAP_SR_REPORT.md**, **JOURNEYMAP_ESG_DATA.md**, **JOURNEYMAP_DASHBOARD.md**를 참조.
