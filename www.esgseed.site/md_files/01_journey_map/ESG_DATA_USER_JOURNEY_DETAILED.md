# 도표 및 그래프(ESG 데이터) 페이지 — 상세 사용자 저니맵

## 1. 개요

본 문서는 **도표 및 그림 생성** 페이지(`/charts`)에서 사용자가 ESG 데이터를 시각화(차트)하고 도표(테이블)를 작성·저장하여 최종보고서에 반영하기까지의 **전체 여정**을 6열 구조로 상세히 기술한 사용자 저니맵입니다.  
GHG·SR 저니맵과 동일한 **행(열) 구조**를 사용하며, **사용자 관점**과 **화면에 보이는 버튼·탭·입력란**만 사용합니다. 시스템/코드 용어는 사용하지 않습니다.

**대상 사용자**: ESG 담당자(팀장·팀원), 보고서 작성 담당자  
**목표**: 보고서에 넣을 차트와 ESG DATA 테이블(도표)을 생성·저장하고, 최종보고서에서 확인·다운로드할 수 있도록 하는 흐름을 이해하고 개선 포인트를 도출한다.

---

## 2. 저니맵 행(열) 구조

| 열 순서 | 열 이름 | 설명 |
|--------|----------|------|
| 1 | **단계 (Step)** | 사용자가 겪는 단계·순서 (예: 1. 진입, 2. ESG 영역 선택) |
| 2 | **사용자 행동 (User Action)** | 해당 단계에서 사용자가 **직접 하는 행동** (클릭, 입력, 선택) |
| 3 | **화면 상태 (Screen State)** | 그때 화면에 **보이는 것** (탭, 카드, 버튼, 입력란, 미리보기, 안내 문구) |
| 4 | **감정/생각 (Emotion/Thought)** | 사용자가 느끼거나 생각할 수 있는 것 |
| 5 | **문제점 (Pain Point)** | 불편함, 혼란, 막힘 |
| 6 | **기회 요소 (Opportunities / Solutions)** | 개선·해결 방향 |

---

## 3. 화면에 보이는 요소 요약

저니맵에서 언급하는 **실제 화면 요소**만 정리합니다.

- **상단 네비**: 홈 · 회사정보 · GHG 산정 · **도표 및 그림 생성** · SR 작성 · 최종보고서
- **페이지 상단**: **Environmental** · **Social** · **Governance** (ESG 탭)
- **Environmental 내**: **온실가스 / 에너지** · **폐기물 / 대기** · **용수 / 폐수** (카테고리 탭)
- **좌측**: **데이터 소스** (드롭다운), **차트 설정** (범례/멀티, 차트 제목, X축/Y축 라벨, **데이터 포인트**·추가·제거)
- **중앙**: **차트 미리보기** (PNG·저장), **ESG DATA 테이블** (테이블 선택, 저장·다운로드·행 추가·관리)
- **우측**: **차트 유형** (막대, 누적 막대, 수평 막대, 원형, 도넛, 선형, 혼합형, 영역), **차트 생성하기**, **차트 갤러리**, **도표 갤러리**
- **메시지**: "차트가 저장되었습니다. 최종보고서 페이지에서 확인할 수 있습니다." 등 토스트

## 4. 상세 저니맵 — 13단계 (6열)

아래 표는 **단계별로 6열**을 채운 상세 시트입니다.

| 단계 (Step) | 사용자 행동 (User Action) | 화면 상태 (Screen State) | 감정/생각 (Emotion/Thought) | 문제점 (Pain Point) | 기회 요소 (Opportunities / Solutions) |
|-------------|---------------------------|---------------------------|-----------------------------|---------------------|----------------------------------------|
| **1. 진입** | 상단 메뉴에서 **도표 및 그림 생성** 클릭 | 도표 및 그림 생성 페이지가 열림. 상단에 **Environmental**·**Social**·**Governance** 탭이 크게 보임. 기본으로 **Environmental** 탭이 활성화되어 **온실가스/에너지**·**폐기물/대기**·**용수/폐수** 카테고리 탭, 좌측 **데이터 소스**·**차트 설정** 카드, 중앙 **차트 미리보기** 영역, 그 아래 **ESG DATA 테이블** 카드, 우측 **차트 유형**·**차트 생성하기**·**차트 갤러리**·**도표 갤러리**가 한 화면에 배치됨 | "보고서에 넣을 차트와 도표를 여기서 만드는구나" | 첫 진입 시 **차트**와 **테이블(도표)** 을 둘 다 쓸지, 어떤 순서로 진행해야 할지 헷갈릴 수 있음 | 페이지 상단 또는 첫 화면에 한 줄 안내: "**차트**: 데이터 소스·유형 선택 후 차트 생성하기 → 저장. **도표**: ESG DATA 테이블에서 셀 입력 후 저장. 둘 다 최종보고서에 반영됩니다." |
| **2. ESG 영역 선택** | 상단 **Environmental** 또는 **Social** 또는 **Governance** 탭 클릭 | 선택한 탭이 강조 표시되고, 해당 영역에 맞는 **카테고리 탭**(Environmental일 때만)·**데이터 소스** 드롭다운 옵션·**차트 유형**·**ESG DATA 테이블** 프리셋만 표시됨. Environmental 선택 시 온실가스/에너지·폐기물/대기·용수/폐수 카테고리와 해당 데이터 소스 목록(Scope 1+2 총 배출량, Scope 1 총 직접 배출량 등)이 보임 | "환경 데이터부터 차트로 만들자" / "사회 지표는 Social에서 하면 되겠다" | 탭 이름이 **영문**(Environmental, Social, Governance)이라 일부 사용자에게 직관이 떨어질 수 있음 | 탭 옆에 툴팁 또는 부제 표시: "Environmental: 온실가스·에너지·폐기물·용수 등", "Social: 인력·안전·협력사·지역사회 등", "Governance: 지배구조·윤리·리스크 등" |
| **3. 카테고리 선택 (Environmental 내)** | **온실가스 / 에너지** 또는 **폐기물 / 대기** 또는 **용수 / 폐수** 탭 클릭 | 선택한 카테고리에 맞는 **데이터 소스** 드롭다운 옵션과 **ESG DATA 테이블** 프리셋만 노출됨. 예: 온실가스/에너지 선택 시 데이터 소스에 "Scope 1+2 총 배출량", "Scope 1 총 직접 배출량", "Scope 2 총 간접 배출량", "Scope 3 총 기타간접", "총 에너지 소비량", "재생에너지 사용 현황" 등이 표시되고, 테이블 선택에는 "온실가스 배출량", "에너지/재생에너지", "투자/PUE" 등이 표시됨 | "온실가스 배출량 차트를 만들겠다" / "용수 사용량은 용수/폐수에서 하면 되겠다" | 카테고리와 **데이터 소스 항목이 많아** 원하는 항목을 빨리 찾기 어려울 수 있음 | 데이터 소스 드롭다운에 **검색** 기능 또는 "자주 쓰는 항목" 상단 고정 옵션 제공 |
| **4. 데이터 소스 및 차트 설정 입력** | **데이터 소스** 드롭다운에서 시각화할 데이터 유형 선택(예: Scope 1 + Scope 2 총 배출량) → **차트 설정**에서 **차트 제목**, **X축 라벨**, **Y축 라벨** 입력 → **데이터 포인트**에 레이블(예: 2021년, 2022년)과 값 입력. 필요 시 **추가**로 행 추가, **제거**로 행 삭제. **범례**·**멀티** 스위치로 멀티 시리즈 사용 시 **시리즈 추가**·**시리즈 제거** | 데이터 소스 선택 시 차트 제목이 해당 소스 라벨로 자동 채워질 수 있음. 데이터 포인트는 레이블·값 쌍 여러 개(예: 최대 10개). **차트 미리보기** 영역에는 아직 "차트 유형과 데이터 소스를 선택한 후 차트 생성하기 버튼을 클릭하세요" 안내 또는 빈 상태가 보임 | "연도별 배출량 숫자만 넣으면 되겠다" / "제목은 자동으로 채워져서 편하네" | 레이블·값을 **잘못 입력했을 때** 수정 방법이 직관적이지 않거나, 데이터 소스마다 **단위**(tCO2e, MWh 등)가 달라 헷갈릴 수 있음 | 데이터 소스별 **권장 단위**를 Y축 라벨 옆에 안내(예: "단위: tCO2e"). 데이터 포인트에 **예시 값** 플레이스홀더 또는 샘플 행 제공 |
| **5. 차트 유형 선택** | 우측 **차트 유형** 카드에서 **막대 차트**, **누적 막대 차트**, **수평 막대 차트**, **원형 차트**, **도넛 차트**, **선형 차트**, **혼합형(막대+선)**, **영역 차트** 중 하나 클릭 | 선택한 유형이 강조 표시(테두리·배경). 각 유형 아래 짧은 설명(예: "카테고리별 데이터 비교에 적합", "시간별 변화 추이 표시에 적합")이 있으면 표시됨 | "연도 추이는 선형이 좋겠다" / "비율은 도넛으로 보여주자" | 데이터 특성과 **맞지 않는 유형**을 골라도 막지 않아, 생성 후에야 부적절함을 느낄 수 있음 | 데이터 소스 또는 **데이터 포인트 개수**에 따라 "추천: 선형 차트" 등 짧은 추천 문구 표시 |
| **6. 차트 생성하기 클릭** | 우측 **차트 생성하기** 버튼 클릭 | 버튼이 **"생성 중..."**으로 바뀌었다가 완료 후 **차트 미리보기** 영역에 차트가 렌더링됨. **PNG**·**저장** 버튼이 활성화됨. 데이터 포인트에 유효한 레이블·값이 없으면 미리보기는 빈 상태 또는 "차트 유형과 데이터 소스를 선택한 후 차트 생성하기 버튼을 클릭하세요" 안내 유지 | "차트가 나왔다" / "의도한 대로 그려졌네" | 데이터 포인트가 **비어 있거나** 레이블만 있고 값이 0이면 차트가 의미 없어 보이거나 빈 화면으로 나올 수 있음 | "레이블과 값이 모두 입력된 행만 차트에 반영됩니다" 안내. 생성 전 **유효성 경고**(예: "값이 입력된 행이 없습니다") 표시 |
| **7. 차트 미리보기 확인 및 PNG 다운로드** | **차트 미리보기**에서 생성된 차트를 확인. 필요 시 **PNG** 버튼 클릭하여 이미지 파일로 다운로드 | 미리보기 카드 안에 차트 캔버스, 상단 **PNG**·**저장** 버튼. PNG 클릭 시 차트 제목 기반 파일명으로 png 파일이 다운로드됨 | "보고서 외부에서도 쓸 수 있게 PNG로 받아두자" / "발표 자료에 넣으려고 이미지로 저장해야지" | **PNG**와 **저장**의 차이(저장 = 최종보고서 포함, PNG = 로컬 파일만)가 처음엔 불명확할 수 있음 | **PNG**·**저장** 버튼에 툴팁: "PNG: 이미지 파일 다운로드", "저장: 최종보고서에 이 차트 포함" |
| **8. 차트 저장(최종보고서에 포함)** | **차트 미리보기** 카드에서 **저장** 버튼 클릭 | "차트가 저장되었습니다. 최종보고서 페이지에서 확인할 수 있습니다." 토스트 메시지 표시. **차트 갤러리**에 방금 저장한 차트 썸네일이 추가됨. 갤러리에서 썸네일 클릭 시 해당 차트 설정·데이터가 다시 로드됨 | "이 차트는 최종보고서 PDF에 들어가겠구나" / "나중에 다시 불러와서 수정할 수 있겠다" | **동일한 설정**으로 여러 번 저장하면 갤러리·최종보고서에 **중복**으로 쌓일 수 있음 | "이미 동일 제목으로 저장된 차트가 있습니다. 덮어쓸까요?" 선택 옵션(기능 확장 시). 또는 저장 시 제목·설정 해시로 중복 여부 체크 후 안내 |
| **9. 차트 갤러리에서 재사용 또는 삭제** | **차트 갤러리**에서 썸네일 클릭 → 해당 차트가 설정·미리보기에 로드되어 수정 후 다시 **차트 생성하기**·**저장** 가능. 또는 썸네일의 **✕** 클릭 → 갤러리에서 삭제 | 갤러리 카드에 저장된 차트 썸네일(예: 최대 4개). 클릭 시 데이터 소스·차트 유형·제목·데이터 포인트가 좌측·우측에 채워짐. ✕ 클릭 시 "차트가 갤러리에서 삭제되었습니다." 토스트 | "저장해 둔 차트를 다시 불러와서 수정할 수 있구나" / "잘못 저장한 건 갤러리에서만 지우면 되겠다" | 갤러리에서 **삭제**해도 이미 **저장**으로 최종보고서에 넣은 차트가 남는지 불명확할 수 있음 | "갤러리에서 삭제해도 이미 최종보고서에 포함된 차트는 유지됩니다" 안내(동작 정책에 따라). 또는 삭제 시 "최종보고서에서도 제거할까요?" 선택 제공 |
| **10. ESG DATA 테이블 선택 및 셀 입력** | **ESG DATA 테이블** 카드에서 **테이블 선택** 드롭다운으로 프리셋 선택(온실가스 배출량, 에너지/재생에너지, 투자/PUE, 용수/폐수, 폐기물/대기) → 표시된 테이블(들)의 각 셀에 값 입력. **행 추가**로 행 추가, 행 끝 **관리(제거)** 버튼으로 행 삭제 | 선택한 프리셋에 따라 Scope 1·2 온실가스 배출량, Scope 3 온실가스, 용수 취수량·방류량, 폐기물 발생량 등 테이블이 카드 형태로 나열됨. 각 테이블에 제목·**저장**·**다운로드**·**행 추가**, 편집 가능한 셀 | "보고서에 넣을 숫자 표를 여기서 채우는구나" / "GHG 산정에서 나온 숫자를 여기 테이블에 옮기면 되겠다" | 테이블이 **많고** 셀이 많아 **어떤 테이블을 먼저** 채워야 할지 모를 수 있음. GHG 산정 결과와 **연동**되는지 처음엔 모를 수 있음 | "GHG 산정 결과를 여기 테이블에 반영할 수 있습니다" 등 **연동 안내**. 테이블별 "권장 작성 순서" 또는 완성도 표시 |
| **11. 도표(테이블) 저장(최종보고서에 포함)** | 각 테이블 카드에서 **저장** 버튼 클릭 | "도표가 최종보고서에 저장되었습니다." 토스트. **도표 갤러리**에 해당 도표가 추가됨. 도표 갤러리에는 "저장된 도표가 없습니다. 각 테이블의 저장을 눌러 추가하세요" 또는 저장된 도표 목록(제목·저장 시점)이 표시됨 | "이 표는 최종보고서에 들어가겠구나" / "차트랑 도표 둘 다 저장해야 최종보고서에 다 들어가는구나" | **같은 테이블**을 여러 번 저장하면 도표 갤러리·최종보고서에 **중복** 저장될 수 있음 | 테이블별 **"마지막 저장 시점"** 표시로 중복 저장 방지 안내. 또는 "이미 저장된 도표가 있습니다. 덮어쓸까요?" 옵션 |
| **12. 테이블 다운로드(CSV)** | 테이블 카드에서 **다운로드** 버튼 클릭 | 테이블 제목 기반 파일명으로 **CSV** 파일이 다운로드됨. "저장"과 별개로 **로컬 백업**용 | "엑셀에서 수정하려면 CSV로 받아두자" / "외부 검토용으로 보낼 때 쓰겠다" | CSV **인코딩·구분자** 안내가 없으면 엑셀에서 **깨져 보이거나** 한글이 깨질 수 있음 | "UTF-8 CSV입니다. 엑셀에서 열 때 데이터 → 텍스트 나누기로 가져오세요" 등 짧은 안내 또는 다운로드 시 툴팁 |
| **13. 완료 및 최종보고서로 이동** | 차트·도표 저장을 마친 후 네비에서 **최종보고서** 클릭하여 PDF 생성·다운로드 | 최종보고서 페이지. 보고서 완성도·포함된 데이터(저장된 차트 N개, 저장된 도표 등) 표시. **PDF 다운로드** 시 저장한 차트·도표가 보고서에 포함됨 | "도표 및 그림 생성에서 만든 차트와 표가 최종보고서에 들어갔구나" / "이제 PDF로 내보내면 된다" | 차트/도표 **저장**과 최종보고서 **PDF 반영**이 같은 세션인지, 이전에 저장한 것이 남는지 직관이 부족할 수 있음 | 최종보고서 페이지에 **"포함된 차트: N개, 도표: N개"** 명시. "도표 및 그림 생성에서 추가하기" 등 **도표 및 그림 생성**으로 이동 링크 제공 |

## 5. 시나리오별 흐름 요약

### 5-1. 차트 중심 흐름 (빠른 경로)

1. 진입 → 2. ESG 영역 선택(Environmental 등) → 3. 카테고리 선택 → 4. 데이터 소스·차트 설정 입력 → 5. 차트 유형 선택 → 6. 차트 생성하기 클릭 → 7. 미리보기 확인·PNG(선택) → 8. 저장 → (선택) 9. 갤러리에서 재사용·삭제
