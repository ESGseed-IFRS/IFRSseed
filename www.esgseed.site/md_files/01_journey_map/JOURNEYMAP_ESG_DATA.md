# 도표 및 그래프(ESG 데이터) 페이지 사용자 저니맵

도표 및 그림 생성(`/charts`) 페이지의 **현재 프론트엔드 화면**을 기준으로 작성한 사용자 저니맵입니다. **행 구조**는 GHG·SR 저니맵과 동일하며, **사용자 관점**과 **화면에 실제로 보이는 버튼·단계**만 사용합니다. 시스템/코드 용어는 사용하지 않습니다.

---

## 1. 세부 시트 행(열) 구조

| 열 순서 | 열 이름 | 설명 |
|--------|----------|------|
| 1 | **단계 (Step)** | 사용자가 겪는 단계·순서 |
| 2 | **사용자 행동 (User Action)** | 해당 단계에서 사용자가 **직접 하는 행동** (클릭, 입력, 선택) |
| 3 | **화면 상태 (Screen State)** | 그때 화면에 **보이는 것** (탭, 카드, 버튼, 입력란, 미리보기) |
| 4 | **감정/생각 (Emotion/Thought)** | 사용자가 느끼거나 생각할 수 있는 것 |
| 5 | **문제점 (Pain Point)** | 불편함, 혼란, 막힘 |
| 6 | **기회 요소 (Opportunities / Solutions)** | 개선·해결 방향 |

---

## 2. 작성 원칙

- **쓸 수 있는 것**: 사용자가 **눈으로 보는 요소** — 상단 탭 이름, 카드 제목, 버튼 문구, 드롭다운·입력란 라벨, 안내 문구. 사용자 **행동** — "Environmental 탭 클릭", "데이터 소스 선택", "차트 생성하기 클릭", "저장 클릭".
- **쓰지 말아야 할 것**: 컴포넌트·함수·변수명, 개발자 전용 용어.

## 3. 현재 화면에 보이는 요소 (프론트엔드 기준)

### 3-1. 상단 네비게이션

| 구분 | 화면에 보이는 이름 |
|------|-------------------|
| 메뉴 | **홈** · **회사정보** · **GHG 산정** · **도표 및 그림 생성** · **SR 작성** · **대시보드** |

### 3-2. 도표 및 그림 생성 페이지 (`/charts`) — 상단

| 구분 | 화면에 보이는 이름 |
|------|-------------------|
| ESG 탭 | **Environmental** · **Social** · **Governance** (큰 버튼, 한 탭 선택 시 해당 영역만 표시) |

### 3-3. Environmental 탭 — 카테고리·데이터·설정

| 구분 | 화면에 보이는 이름 |
|------|-------------------|
| 카테고리 탭 | **온실가스 / 에너지** · **폐기물 / 대기** · **용수 / 폐수** |
| 좌측 카드 | **데이터 소스** — "시각화할 데이터 유형을 선택하세요", 드롭다운(플레이스홀더: "데이터 유형 선택") |
| | **차트 설정** — "차트의 세부 설정을 조정하세요" |
| | **범례/멀티시리즈** — "누적 막대/혼합형(막대+선) 및 범례 표시를 위한 멀티 시리즈 입력", **범례** 스위치, **멀티** 스위치 |
| | **차트 제목**, **X축 라벨**, **Y축 라벨** (입력란) |
| | **데이터 포인트** — 레이블·값 입력 행(1. 2. 3. …), **추가**·**제거** 버튼. 멀티 시 **시리즈 추가**·**시리즈 제거**, 시리즈별 이름·타입(막대/선) 입력 |
| 데이터 소스 예시 | 온실가스/에너지: Scope 1 + Scope 2 (총 직/간접 온실가스 배출량), Scope 1 (총 직접 배출량), Scope 2 (총 간접 배출량), Scope 3 (총 기타간접 배출량), 총 에너지 소비량, 재생에너지 사용 현황 등. 폐기물/대기: 총 폐기물 발생량, 화학물질 배출량, 폐기물 현황(처리 유형별/종류별), NOx/SOx/먼지/VOC 배출량 등. 용수/폐수: 용수 사용량, 폐수 처리량, 용수 취수량, 용수 재활용량 등 |

### 3-4. 차트 미리보기·테이블·우측

| 구분 | 화면에 보이는 이름 |
|------|-------------------|
| 중앙 카드 | **차트 미리보기** — "생성된 차트를 확인하고 다운로드하세요" · **PNG** 버튼 · **저장** 버튼 |
| | 미리보기 전: "차트 유형과 데이터 소스를 선택한 후 차트 생성하기 버튼을 클릭하세요" 안내 |
| 테이블 영역 | **ESG DATA 테이블** — "테이블 세트를 선택하고 각 셀 값을 직접 입력/편집할 수 있습니다." |
| | **테이블 선택** 드롭다운: 온실가스 배출량, 에너지/재생에너지, 투자/PUE, 용수/폐수, 폐기물/대기 (카테고리 탭에 따라 옵션 변경) |
| | 각 테이블 카드: 제목·설명(note)·**저장**·**다운로드**·**행 추가**, 셀 입력, 행 끝 **관리** 버튼(행 제거) |
| 우측 카드 | **차트 유형** — "데이터에 적합한 차트 유형을 선택하세요" |
| | 막대 차트, 누적 막대 차트, 수평 막대 차트, 원형 차트, 도넛 차트, 선형 차트, 혼합형(막대+선), 영역 차트 (각각 짧은 설명 문구 포함) |
| 버튼 | **차트 생성하기** (비활성: 차트 유형 또는 데이터 소스 미선택 시), 클릭 시 **생성 중...** 표시 |
| | **차트 갤러리** — "최근 생성된 차트들을 확인하고 재사용하세요", 썸네일 클릭 시 로드, **✕** 삭제. 비어 있으면 "비어있음" 4칸 |
| | **도표 갤러리** — "저장한 ESG DATA 테이블을 최종보고서에 포함할 수 있습니다.", 저장된 도표 목록 또는 "저장된 도표가 없습니다. 각 테이블의 "저장"을 눌러 추가하세요." |

## 4. 단계별 상세 저니맵 (6열)

| 단계 (Step) | 사용자 행동 (User Action) | 화면 상태 (Screen State) | 감정/생각 (Emotion/Thought) | 문제점 (Pain Point) | 기회 요소 (Opportunities / Solutions) |
|-------------|---------------------------|---------------------------|-----------------------------|---------------------|----------------------------------------|
| **1. 진입** | 상단 메뉴에서 **도표 및 그림 생성** 클릭 | 도표 및 그림 생성 페이지. 상단 **Environmental**·**Social**·**Governance** 탭. 기본 **Environmental** 활성 시 **온실가스/에너지**·**폐기물/대기**·**용수/폐수** 카테고리 탭, 좌측 **데이터 소스**·**차트 설정**, 중앙 **차트 미리보기**, 하단 **ESG DATA 테이블**, 우측 **차트 유형**·**차트 생성하기**·**차트 갤러리**·**도표 갤러리** | "보고서에 넣을 차트와 도표를 여기서 만드는구나" | 차트와 테이블(도표)을 둘 다 쓸지, 순서가 어떤지 헷갈릴 수 있음 | "차트: 데이터 소스·유형 선택 후 차트 생성하기 → 저장. 도표: ESG DATA 테이블에서 셀 입력 후 저장. 둘 다 대시보드·최종보고서에 반영됩니다" 한 줄 안내 |
| **2. ESG 영역 선택** | **Environmental** 또는 **Social** 또는 **Governance** 탭 클릭 | 선택한 탭이 강조 표시. 해당 영역의 카테고리 탭·데이터 소스·테이블 프리셋만 표시. Environmental 선택 시 온실가스/에너지·폐기물/대기·용수/폐수와 해당 데이터 소스 목록 | "환경 데이터부터 차트로 만들자" | 탭 이름이 영문이라 일부 사용자에게 직관이 떨어질 수 있음 | 툴팁 또는 부제: "Environmental: 온실가스·에너지·폐기물·용수 등" |
| **3. 카테고리 선택 (Environmental)** | **온실가스 / 에너지** 또는 **폐기물 / 대기** 또는 **용수 / 폐수** 탭 클릭 | 선택한 카테고리에 맞는 **데이터 소스** 드롭다운 옵션과 **테이블 선택** 옵션만 노출. 예: 온실가스/에너지 시 데이터 소스에 Scope 1+2, Scope 1, Scope 2, Scope 3, 총 에너지 소비량, 재생에너지 사용 현황 등. 테이블 선택에 온실가스 배출량, 에너지/재생에너지, 투자/PUE | "온실가스 배출량 차트를 만들겠다" | 카테고리·데이터 소스가 많아 원하는 항목을 빨리 찾기 어려울 수 있음 | 데이터 소스 드롭다운 검색 또는 자주 쓰는 항목 상단 고정 |
| **4. 데이터 소스 및 차트 설정 입력** | **데이터 소스** 드롭다운에서 데이터 유형 선택(예: Scope 1 + Scope 2) → **차트 설정**에서 **차트 제목**, **X축 라벨**, **Y축 라벨** 입력 → **데이터 포인트**에 레이블·값 입력. **추가**·**제거**로 행 수 조절. **범례**·**멀티** 스위치 사용 시 **시리즈 추가**·**시리즈 제거** | 데이터 소스 선택 시 차트 제목이 해당 소스 라벨로 자동 채워질 수 있음. 데이터 포인트는 레이블·값 쌍(최대 10개). 차트 미리보기 영역에는 "차트 유형과 데이터 소스를 선택한 후 차트 생성하기 버튼을 클릭하세요" 또는 빈 상태 | "연도별 배출량 숫자만 넣으면 되겠다" | 레이블·값 수정이 직관적이지 않거나, 데이터 소스마다 단위가 달라 헷갈릴 수 있음 | 데이터 소스별 권장 단위 안내(Y축 라벨 옆), 예시 값 플레이스홀더 |
| **5. 차트 유형 선택** | 우측 **차트 유형** 카드에서 **막대 차트**, **누적 막대 차트**, **수평 막대 차트**, **원형 차트**, **도넛 차트**, **선형 차트**, **혼합형(막대+선)**, **영역 차트** 중 하나 클릭 | 선택한 유형이 강조(테두리·배경). 각 유형 아래 "카테고리별 데이터 비교에 적합", "시간별 변화 추이 표시에 적합" 등 설명 | "연도 추이는 선형이 좋겠다" | 데이터 특성과 맞지 않는 유형을 골라도 막지 않아 생성 후 부적절함을 느낄 수 있음 | 데이터 소스·포인트 개수에 따라 "추천: 선형 차트" 등 짧은 추천 문구 |
| **6. 차트 생성하기 클릭** | 우측 **차트 생성하기** 버튼 클릭 | 버튼이 **생성 중...**으로 바뀌었다가 완료 후 **차트 미리보기** 영역에 차트 렌더링. **PNG**·**저장** 버튼 활성화. 레이블·값이 없으면 미리보기 빈 상태 또는 안내 유지 | "차트가 나왔다" | 데이터 포인트가 비어 있거나 값이 0이면 차트가 의미 없어 보일 수 있음 | "레이블과 값이 모두 입력된 행만 차트에 반영됩니다" 안내, 생성 전 유효성 경고 |
| **7. 차트 미리보기 확인 및 PNG 다운로드** | **차트 미리보기**에서 생성된 차트 확인. **PNG** 버튼 클릭 | 미리보기 카드 안에 차트, **PNG**·**저장** 버튼. PNG 클릭 시 차트 제목 기반 파일명으로 png 다운로드 | "보고서 외부에서도 쓸 수 있게 PNG로 받아두자" | PNG와 "저장"의 차이(저장=대시보드·보고서 포함, PNG=로컬 파일만)가 처음엔 불명확할 수 있음 | "PNG: 이미지 파일 다운로드. 저장: 대시보드·최종보고서에 이 차트 포함" 툴팁 |
| **8. 차트 저장** | **차트 미리보기** 카드에서 **저장** 버튼 클릭 | "차트가 저장되었습니다." / "대시보드에서 확인할 수 있습니다." 토스트. **차트 갤러리**에 방금 저장한 차트 썸네일 추가. 썸네일 클릭 시 해당 차트 설정·데이터 로드 | "이 차트는 대시보드·보고서에 들어가겠구나" | 동일 설정으로 여러 번 저장하면 갤러리·보고서에 중복으로 쌓일 수 있음 | "이미 동일 제목으로 저장된 차트가 있습니다. 덮어쓸까요?" 선택 옵션(기능 확장 시) |
| **9. 차트 갤러리에서 재사용 또는 삭제** | **차트 갤러리**에서 썸네일 클릭 → 설정·미리보기에 로드되어 수정 후 다시 생성·저장. 또는 썸네일 **✕** 클릭 | 갤러리 카드에 저장된 차트 썸네일(최대 4개). 클릭 시 데이터 소스·차트 유형·제목·데이터 포인트 채워짐. ✕ 클릭 시 "차트가 갤러리에서 삭제되었습니다." 토스트 | "저장해 둔 차트를 다시 불러와서 수정할 수 있구나" | 갤러리에서 삭제해도 이미 저장으로 보고서에 넣은 차트가 남는지 불명확할 수 있음 | "갤러리에서 삭제해도 이미 보고서에 포함된 차트는 유지됩니다" 안내 |
| **10. ESG DATA 테이블 선택 및 셀 입력** | **ESG DATA 테이블**에서 **테이블 선택** 드롭다운으로 프리셋 선택(온실가스 배출량, 에너지/재생에너지, 투자/PUE, 용수/폐수, 폐기물/대기) → 표시된 테이블(들)의 각 셀에 값 입력. **행 추가**로 행 추가, 행 끝 **관리** 버튼으로 행 삭제 | 선택한 프리셋에 따라 Scope 1·2 온실가스 배출량, Scope 3 온실가스, 용수 취수량·방류량, 폐기물 발생량 등 테이블이 카드로 나열. 각 테이블 제목·**저장**·**다운로드**·**행 추가**, 편집 가능한 셀 | "보고서에 넣을 숫자 표를 여기서 채우는구나" | 테이블이 많고 셀이 많아 어떤 테이블을 먼저 채워야 할지 모를 수 있음 | "GHG 산정 결과를 여기 테이블에 반영할 수 있습니다" 등 연동 안내 |
| **11. 도표(테이블) 저장** | 각 테이블 카드에서 **저장** 버튼 클릭 | "도표가 최종보고서에 저장되었습니다." 토스트. **도표 갤러리**에 해당 도표 추가. 도표 갤러리에 "저장된 도표가 없습니다. 각 테이블의 "저장"을 눌러 추가하세요." 또는 저장된 도표 목록 | "이 표는 최종보고서에 들어가겠구나" | 같은 테이블을 여러 번 저장하면 도표 갤러리·보고서에 중복 저장될 수 있음 | 테이블별 "마지막 저장 시점" 표시, 중복 저장 방지 안내 |
| **12. 테이블 다운로드(CSV)** | 테이블 카드에서 **다운로드** 버튼 클릭 | 테이블 제목 기반 파일명으로 CSV 다운로드. 저장과 별개로 로컬 백업용 | "엑셀에서 수정하려면 CSV로 받아두자" | CSV 인코딩·구분자 안내가 없으면 엑셀에서 깨져 보일 수 있음 | "UTF-8 CSV입니다. 엑셀에서 열 때 데이터 → 텍스트 나누기로 가져오세요" 안내 |
| **13. 완료 및 대시보드·보고서로 이동** | 차트·도표 저장을 마친 후 네비에서 **대시보드** 클릭하여 진행 현황·최종보고서 확인 | 대시보드 페이지. 보고서 완성도·포함된 데이터(저장된 차트·도표) 표시. PDF 다운로드 시 저장한 차트·도표가 보고서에 포함됨 | "도표 및 그림 생성에서 만든 차트와 표가 보고서에 들어갔구나" | 차트/도표 저장과 최종보고서 PDF 반영이 같은 세션인지, 이전 저장이 남는지 직관이 부족할 수 있음 | 대시보드·최종보고서에 "포함된 차트: N개, 도표: N개" 명시, 도표 및 그림 생성으로 이동 링크 |

## 6. 요약

- **행 구조**: 단계 | 사용자 행동 | 화면 상태 | 감정/생각 | 문제점 | 기회 요소 (6열 고정).
- **기준**: 현재 프론트엔드(`/charts`, Environmental/Social/Governance 페이지)에 **실제 노출된** 탭·카드 제목·버튼·드롭다운·입력란·토스트 문구를 반영함.
- **내용**: §3의 화면 요소 목록과 §4의 단계별 상세 표를 기준으로, 각 단계마다 6열을 채워 세부 시트로 활용할 수 있음.
- **네비·토스트**: 상단 메뉴에는 **대시보드**가 있으며, 차트 저장 시 "대시보드에서 확인할 수 있습니다." 문구가 사용됨. 최종보고서 반영은 대시보드·최종보고서 플로우와 연계됨.

이 문서는 현재 구현된 도표 및 그래프(ESG 데이터) 페이지 화면을 참고하여 작성한 사용자 저니맵입니다.
