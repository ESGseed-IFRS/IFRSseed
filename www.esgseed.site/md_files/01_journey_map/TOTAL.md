새 문서에서 만들 수 있게, “상세 v7.0 한글 저니맵”에 필요한 **최소 노드/구성 목록(복붙용)**을 정리해드릴게요.

1) 다이어그램 구조(권장)
스윔레인 2개: ESG 팀장(결재권자) / ESG 팀원
중앙 허브: 대시보드(허브)
모듈(허브와 양방향 연결): 회사정보 / GHG 산정(/ghg-calculation) / SR 작성(/report) / 도표·그림(/charts) / 보고서 출력
의사결정(다이아몬드) 4개: 회사 미등록? / 팀장 승인 완료? / 권한 있음? / 스코어카드 4항목 충족?
범례(한국어): 시작/종료, 프로세스, 의사결정, 팀장 전용, 팀원 전용
2) 노드 텍스트(페이지별, 팀장/팀원)
A. 로그인·회원가입
(팀장) 회사 등록(최초) → DART 자동 불러오기 → 초대 링크 발송 → 팀장 등록(초대 링크) → 비밀번호 설정 → 대시보드
(팀원) 회원가입 → 회사 확인 → [회사 미등록?](예: “회사 등록 먼저 필요”) → 이메일 인증 → 비밀번호 설정 → HRIS 조회/역할 자동할당 또는 승인 대기 → [팀장 승인 완료?] → 로그인/대시보드
B. 대시보드(허브)
(팀장) 전체 진행률 · 마감일/D-day · 확인 필요/다음 액션 · 페이지별 카드(담당자/이동) · 팀원 초대 · 가입 승인/반려 · 팀원 현황 · 스코어카드(회사정보 제출/GHG 저장/SR≥90/차트 저장) · 최종 승인 · 활동 로그 · Excel/PPT 다운로드
(팀원) 전체 진행률 · 확인 필요/다음 액션 · 페이지별 카드 · 팀장 피드백(수정요청/승인) · 수정 후 재요청 · (최종 승인 버튼 없음) · 팀장 최종 승인 후 다운로드
C. 회사정보
(팀장) DART+수기 입력/수정 · 4단계(진입/레이아웃/입력·확인/시스템 연동 조회) · 검증 · 제출
(팀원) 조회 중심 · 담당 섹션만 제한 수정(가능 시) · 시스템 연동 조회 · 수정 필요 시 팀장에게 요청
D. GHG 산정 (/ghg-calculation)
(공통 요약) 3탭: 산정·공시 | 감사·검증 대응 | 리포트 생성
(공통 요약) 6입력탭: 전력·연료·냉매·폐기물·물류·원료
(공통 요약) 전력/연료: Step1 데이터 확인 → Step2 산정 실행 → Step3 결과 확인 → 저장
(팀원) 전사 GHG 데이터 입력·저장: 전체 6탭 접근 · EMS/엑셀 업로드 · Step1~3 수행 · 전체 결과 저장 · CSV 내보내기 · 계수 기준연도 입력
(팀장) 검토·승인·리포트: 팀원 입력 결과 검토 · 감사·검증 대응(체크리스트/데이터계보/증빙패키지 내보내기) · 리포트 생성(PDF/Excel/일괄) · 결과 확정·승인
E. SR 작성 (/report)
(공통) 목차 선택 · 준수율 확인 · 시각화 추천 · 도표 생성 이동
(팀원) 직접 작성/AI 문단 생성 · 저장 · 담당 섹션 작성/수정 · 팀장 피드백 반영 후 재요청
(팀장) 전체 준수율/진행 모니터링 · 팀원 작성 내용 검토 · 피드백/수정 요청 · 승인
F. 도표·그림 (/charts)
(공통) E/S/G 탭 · 데이터 소스/차트 유형 · 차트 생성하기 · ESG DATA 테이블 입력/저장 · 갤러리 · 최종보고서 반영
(팀원) 차트·도표 생성·저장 · 갤러리 관리 · 최종보고서 반영 요청 또는 반영 확인
(팀장) 팀원이 저장한 차트·도표 반영 여부 검토 · 스코어카드 충족 확인 · 필요 시 승인·반영 지시
G. 보고서 출력
(팀장) [스코어카드 4항목 충족?] → 최종 승인 → Excel(.xlsx)/PPT(.pptx) 다운로드
(팀원) [팀장 최종 승인 완료?] → 다운로드
원하시면, 새 문서가 준비되면 위 목록 그대로 사용해서 제가 한글 스윔레인 다이어그램으로 생성해드릴게요.

스윔레인 1: ESG 팀장(결재권자)
A. 로그인·회원가입
회사 등록(최초)
DART 자동 불러오기(기본 회사정보 자동 채움)
팀원 초대를 위한 초대 링크 발송
팀장 등록(초대 링크) 및 계정 생성
비밀번호 설정
로그인
이후 이용 시: 이메일/비밀번호 또는 소셜 로그인 → 대시보드·권한별 메뉴
B. 대시보드(허브) (/dashboard)
전체 진행률·마감일·D-day 확인
페이지별 진행 상태 카드 확인(회사정보·GHG 산정·SR 작성·도표 생성) — 담당자 표시·이동 →
확인 필요 사항·다음 액션 목록 확인 후 해당 페이지로 이동
팀원 현황: 가입 승인 대기 건에 대해 승인/반려
+ 팀원 초대
팀원 목록 확인(이름·부서·담당 페이지·진행률·활성/승인 대기)
보고서 완성도 스코어카드 확인(회사정보 제출·GHG 저장·SR 90%·차트 저장)
최근 활동 로그 확인
C. 회사정보 (/company-info)
회사정보 진입(대시보드 또는 메뉴)
DART 연동 데이터 확인 + 필요 항목 수기 입력/수정(연락처·ESG 목표·이해관계자 등)
4단계 흐름 수행(진입 → 레이아웃 확인 → 입력·확인 → 시스템 연동 조회)
시스템 연동 카드에서 적재된 데이터 조회
정보 확인 및 검증(필수값/형식/일관성)
최종 보고서에 제출(이후 모듈 기준 데이터로 사용)
D. GHG 산정 (/ghg-calculation)
팀원이 입력·저장한 전사 GHG 데이터 검토
감사·검증 대응 탭: 요건 체크리스트·데이터 계보 추적·증빙 패키지 내보내기
리포트 생성 탭: PDF Report·Excel Details·일괄 다운로드
결과 확정·승인(대시보드 스코어카드 등과 연계)
E. SR 작성 (/report)
전체 준수율/진행 상황 모니터링
팀원 작성 내용 검토
피드백 또는 수정 요청
승인(필요 시)
F. 도표·그림 생성 (/charts)
팀원이 저장한 차트·도표 반영 여부 검토
스코어카드 충족 여부 확인(차트·도표 저장)
필요 시 승인·반영 지시
G. 보고서 출력 (대시보드 내)
의사결정: 스코어카드 4항목 충족?(회사정보 제출·GHG 저장·SR≥90%·차트 저장)
Yes: 최종 승인 버튼 클릭 → Excel(.xlsx)·PowerPoint(.pptx) 다운로드
No: 미충족 항목 해당 페이지로 이동하여 보완 후 재확인

스윔레인 2: ESG 팀원
A. 로그인·회원가입
회원가입
회사 확인(회사명·사업자번호 입력 후 회사정보 자동 불러오기)
의사결정: 회사 미등록?
Yes: 회사 등록 먼저 필요(종료)
No: 이메일 인증 → 비밀번호 설정 → HRIS 조회/역할 자동할당 또는 승인 대기
의사결정: 팀장 승인 완료?
Yes: 로그인(다음 단계로 진행)
No: 승인 대기(알림/재요청 등 후속 절차)(종료)
이후 이용 시: 이메일/비밀번호 또는 소셜 로그인 → 권한별 메뉴
B. 대시보드(허브) (/dashboard)
전체 진행률·페이지별 진행 상태 카드 확인(담당자 표시 없음)
확인 필요 사항·다음 액션 목록에서 본인 담당 항목 클릭 → 해당 페이지로 이동
팀장 피드백 확인(수정 요청·승인 완료)
수정 요청 시 해당 페이지에서 수정 후 재요청
보고서 완성도 스코어카드·섹션별 진행 확인(최종 승인 버튼 비표시)
C. 회사정보 (/company-info)
회사정보 조회 중심으로 확인
담당 섹션만 제한 수정(허용된 범위 내)
시스템 연동 카드에서 적재된 데이터 조회
수정 필요 시 팀장에게 요청(권한 제한 또는 승인 필요)
D. GHG 산정 (/ghg-calculation)
전사 GHG 데이터 입력·저장(전체 6탭 접근)
공시 프레임워크 선택(ISSB/KSSB/K-ETS/GRI/ESRS)
전력·연료 탭: STEP 1 데이터 확인 → STEP 2 산정 실행 → STEP 3 결과 확인 → 이 탭 결과 저장
EMS 불러오기·엑셀 업로드로 데이터 수집/적재
전체 결과 저장
필요 시 CSV 내보내기
계수 기준 연도(2024/2023 KR Emission Factors) 입력·관리
감사·검증 대응 탭: 요건 체크리스트·데이터 계보 등 조회(리포트 다운로드는 팀장 또는 권한 시)
E. SR 작성 (/report)
목차(Table of Contents) 선택
직접 작성 또는 AI 문단 생성 후 저장
준수율(컴플라이언스) 확인
시각화 추천 확인·도표 생성으로 이동(필요 시)
담당 섹션 작성/수정
팀장 피드백 반영 후 재요청(승인/확인 요청)
F. 도표·그림 생성 (/charts)
Environmental/Social/Governance 탭 선택
데이터 소스 및 차트 유형 선택
차트 생성하기 → 저장
ESG DATA 테이블 입력/저장(필요 시)
차트 갤러리·도표 갤러리에서 관리
최종보고서 반영 확인(필요 시 팀장에게 반영 요청)
G. 보고서 출력 (대시보드 내)
의사결정: 팀장 최종 승인 완료?
Yes: Excel(.xlsx)·PowerPoint(.pptx) 다운로드
No: 스코어카드·섹션별 진행만 확인(다운로드 버튼 비활성)
