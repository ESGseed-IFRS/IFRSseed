# SR 보고서 작성용 ERP 데이터 가이드

> **목적**: SR(지속가능성 보고서) 작성 시 ERP에서 가져올 수 있는 데이터를 정리하고, SR 목차·공시기준별로 매핑한다.  
> **대상**: ESG/지속가능경영 담당자, SR 작성자  
> **기준**: GRI, KSSB, IFRS S1/S2, ESRS, SASB

---

## 1. 개요

SR 보고서는 환경(E), 사회(S), 지배구조(G) 영역의 정량·정성 데이터를 요구한다. ERP는 이 중 상당수를 **원시 데이터 출처**로 제공할 수 있다.  
본 문서는 **ERP에서 끌어올 수 있는 데이터**를 SR 목차별·공시기준별로 정리한다.

### 1.1 ERP 모듈 ↔ SR 영역 대응

| SR 영역 | 해당 SR 페이지(예) | 주요 ERP 모듈 |
|---------|-------------------|---------------|
| **Environmental** | 온실가스(36), 기후대응(41), 수자원(55), 폐기물(61) | 구매·재고·생산·유틸리티·물류·설비 |
| **Social** | 임직원(70), 안전보건(79), 협력회사(85), 출장·통근 | HR·경비·구매·물류 |
| **Governance** | 지배구조(105), 리스크(112), 윤리·준법(114~116) | 별도 시스템(법무·감사) 연동 필요 |

> **참고**: Governance 관련 데이터는 ERP 외에 법무·감사·리스크 관리 시스템과의 연동이 일반적이다.

## 2. 카테고리별 ERP-SR 통합 요약표

| 카테고리 | ERP에서 가져올 수 있는 대표 데이터 | SR 보고서(GRI / ISSB / ESRS) 연계 지표 예시 | 추출 난이도 | 비고 / 소스 모듈 예시 |
|----------|-----------------------------------|---------------------------------------------|-------------|------------------------|
| 에너지 사용량 | 전력·가스·연료 구매/소비량, 전기요금 내역, 설비별 사용량 (IoT 연동 시 실시간) | Scope 1 & 2 온실가스 배출량, 에너지 집약도 | ★★ | 구매/재무 + MES/WMS 연동 |
| 온실가스 배출 | Scope 1(직접), Scope 2(전력), 일부 Scope 3(구매 원부자재 금액/물량 기반 추정) | GHG 배출 총량, 감축률, 탄소집약도 | ★★★ | 구매 + 재무 + LCA/PCF 모듈 (최근 추가) |
| 용수 사용량 | 공정용수·생활용수 사용량, 폐수 배출량 | Water withdrawal / consumption / discharge | ★★~★★★ | 생산/환경 모듈, 계량기 데이터 연동 |
| 폐기물 발생·처리 | 산업폐기물·일반폐기물 발생량, 재활용/소각/매립 비율, 폐기물 처리 비용 | Waste generated, diverted from disposal, 재활용률 | ★★ | 생산/물류/WMS 모듈 |
| 원부자재 구매 | 구매 품목별 금액·물량·공급처, 원재료 종류 (플라스틱, 금속 등) | Scope 3 Category 1 (Purchased goods), 재생원료 사용 비율 | ★★ | 구매/자재관리 모듈 |
| 제품 탄소발자국 | BOM + 생산량 + 에너지 투입량 → PCF 자동 산출 (최신 ERP에서 지원) | 제품별 탄소배출량 (Product Carbon Footprint) | ★★★★ | 생산 + PCF/LCA 전용 모듈 (SAP 등) |
| 인사·근태·안전 | 임직원 수, 성별·연령·직급 구성, 근태·초과근무, 산업재해 발생 건수·율, 안전교육 이수 시간 | 직원 다양성, LTIFR, TRIR | ★ | HCM / 인사 / 안전보건 모듈 |
| 공급망 관리 | 협력사 수, 구매 금액 비중, 공급사 평가 점수, 현장 안전 점검 이력 | 공급망 ESG 평가, 협력사 안전·인권 리스크 | ★★★ | SRM / 구매 모듈 |
| 재무·투자 | ESG 관련 투자·비용 (재생에너지 설비, 안전시설 등), 안전보건 예산 집행액 | CapEx / OpEx for sustainability, 녹색투자 비율 | ★ | 재무 / 원가관리 모듈 |
| 운송·물류 | 물류 운송량(t·km), 차량별 연료 사용, 운송 거리 | Scope 3 Category 4 & 9 (Upstream/Downstream transportation) | ★★★ | TM / WMS / TMS 연동 |

> **추출 난이도**: ★ 낮음 ~ ★★★★★ 높음

## 13. ERP 연동 시 유의사항

### 13.1 데이터 정합성

- **기간 일치**: SR 회계연도·보고기간과 ERP 데이터 추출 기간 일치
- **사업장 매핑**: K-IFRS 연결실체·KSSB 보고단위와 ERP 사업장 코드 매핑
- **단위·환산**: ERP 단위(m³, L, kWh 등) → 공시 요구 단위로 통일

### 13.2 출처 표시

SR 보고 시 **데이터 출처(ERP vs 수동)** 를 명시하면 공시 신뢰도와 검증 대응에 유리하다.

| 출처 | 표시 예 | 비고 |
|------|---------|------|
| ERP | `source: erp` | 자동 연동 |
| 수동 입력 | `source: manual` | Excel·수기 입력 |

### 13.3 버전·감사 대응

- GHG 배출계수 적용 시 **계수 ID·버전·스냅샷 값** 저장 권장 (Limited Assurance 대응)
