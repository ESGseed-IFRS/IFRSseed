# 검증·증빙 자료 다운로드 패키지 구성 가이드

> **목적**: 제3자 검증·감사 대응 시, 산정 근거가 담긴 증빙 자료를 **미리 패키지로 구성**하여 다운로드하거나 제출용으로 준비할 수 있도록 권장 구조와 팁을 정리함.

---

## 1. 권장 패키지 구조 예시

```
[회사명]_검증증빙자료_[연도]/
├── 01_에너지사용량/
│   ├── 전기_고지서_월별/
│   │   ├── 2024-01_전력사용량.pdf
│   │   ├── 2024-02_전력사용량.pdf
│   │   └── ...
│   ├── 가스_청구서_월별/
│   │   └── ...
│   └── 자체계량기_판독기록.xlsx
├── 02_폐기물반출/
│   ├── 폐기물_반출전표_월별/
│   └── 폐기물_성상별_집계.xlsx
├── 03_공정배출/
│   └── 원료투입_생산량_월별.xlsx
├── 04_Scope3_기타/
│   └── 출장_교통_실적.xlsx
├── 05_산정근거/
│   ├── [회사명]_온실가스_산정파일_[연도].xlsx   ← 월별 시트 포함
│   └── 사용_배출계수_목록.pdf
└── 06_관리문서/
    └── 모니터링_절차서_v1.0.pdf
```

## 2. 폴더별 역할 및 권장 내용

| 폴더 | 역할 | 권장 파일 |
|------|------|-----------|
| **01_에너지사용량** | Scope 1·2 활동자료의 원본 | 전기/가스/연료 청구서, 자체 계량기 판독 기록 |
| **02_폐기물반출** | Scope 3 Cat.12 등 폐기 관련 | 반출 전표, 성상별 집계표 |
| **03_공정배출** | Scope 1 공정 연소·공정 배출 | 원료 투입량, 생산량 월별 현황 |
| **04_Scope3_기타** | Scope 3 나머지 카테고리 | 구매·운송·출장·통근·사용단계 등 실적 |
| **05_산정근거** | 산정 과정·배출계수 | 산정 엑셀, 배출계수 출처 목록 |
| **06_관리문서** | 절차·체계 증빙 | 모니터링 절차서, 검증 절차서 |

---

## 3. 먼저 다운로드/열람 권장 순서

> 검증자·감사인이 **먼저 보고 싶어 하는 자료** 우선순위.

1. **05_산정근거** → 산정 파일·배출계수
   - 전체 배출량 계산 로직과 결론을 먼저 파악
2. **01_에너지사용량** → 청구서·판독기록
   - Scope 1·2 활동자료의 원본·출처 확인
3. **03_공정배출** → 원료·생산량
   - 공정 배출(산업 공정 배출) 근거 확인
4. **02_폐기물반출** → 폐기 관련
   - Scope 3 Cat.12 등 폐기물 관련 자료
5. **04_Scope3_기타** → 구매·운송·출장 등
   - 나머지 Scope 3 카테고리 증빙
6. **06_관리문서** → 절차·체계
   - 모니터링·검증 절차 일관성 확인용

## 4. 팁 & 주의사항

### 4.1 파일 명명

- `YYYY-MM_항목명.pdf` 형식 권장 (예: `2024-01_전력사용량.pdf`)
- 한글·영문 혼용 시 검색·정렬을 위해 일관된 규칙 적용

### 4.2 개인정보·영업비밀

- 청구서 등에 개인정보가 포함되면 **마스킹 후 첨부** 또는 별도 보안 채널로 전달
- 영업비밀 구간은 **요약본·범위만** 제공하거나 비밀유지합의(NDA) 후 제출

### 4.3 원본 보존

- 패키지는 **원본의 복사본**만 포함
- 원본은 별도 보관(예: 5년) 후 폐기 정책에 따라 관리

### 4.4 버전 관리

- 산정 파일은 **버전·작성일** 명시 (예: `산정파일_2024_v2_20250315.xlsx`)
- 절차서도 `v1.0`, `v1.1` 등 버전 표기 유지

### 4.5 데이터 품질 표시

- 산정 시 **실측/추정/공급자제공** 구분 기록
- 추정 시 방법·가정을 산정 근거 문서에 명시

## 5. 플랫폼 연동 시 권장 동작

| 기능 | 권장 |
|------|------|
| **다운로드 전 미리보기** | 패키지 구조·폴더별 파일 목록을 먼저 표시 |
| **선택 다운로드** | 01~06 폴더 중 원하는 폴더만 ZIP 다운로드 |
| **순차 구성** | 위 3절 “먼저 열람 권장 순서”대로 폴더 번호 부여 |
| **산정 파일 포함** | GHG 산정 엑셀·PDF를 `05_산정근거`에 자동 포함 |
| **청구서·전표 연동** | 업로드된 원본을 `01_에너지사용량` 등에 매핑 저장 |
