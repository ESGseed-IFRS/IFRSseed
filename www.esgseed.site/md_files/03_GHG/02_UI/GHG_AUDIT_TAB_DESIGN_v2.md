# GHG 감사·검증 대응 탭 — 화면 설계 스펙 v2

> **목적**: IFRS S2를 중심으로 GRI·KSSB·K-ETS 등 다중 공시기준 대응이 가능한 감사 대응 탭 설계  
> **현재 화면 문제**: 내부통제 지표 중심 → 감사인 관점 IFRS S2 요건 체크 중심으로 전환 필요  
> **관련 문서**: `GHG_AUDIT_TAB_STRATEGY.md`, `GHG_TAB_DESIGN_v2.md`, `DUMMY_DATA_PLANNING.md`

---

## 1. 전체 구조 재설계

### 현재 구조 (문제)

```
왼쪽 사이드 메뉴
├── 내부통제 요약 대시보드   ← 담당자 내부용. 감사 대응용이 아님
├── 데이터 마감/짝정 관리
├── 배출량별 감사 추적
│   ├── 활동자료(Activity Data)
│   ├── 배출계수(Emission Factor)
│   └── 산정결과(Calculated Emissions)
├── 변경 이력(Audit Trail)
├── 증빙자료(Evidence & Integrity)
├── 산정 방법론 및 로직 계보
├── 감사인 전용 뷰
└── 감사 대응 패키지(Export)
```

### 목표 구조

```
상단: 공시 프레임워크 전환 탭
  [IFRS S2]  [GRI]  [KSSB]  [K-ETS]  [ESRS]

메인: 선택된 프레임워크 기준 요건 체크리스트 (항상 첫 화면)

왼쪽 사이드 메뉴
├── 요건 체크리스트        ← 메인. 프레임워크별 자동 전환
├── 데이터 계보 추적       ← 숫자 클릭 시 원천 레코드 드릴다운
├── 수동 조정 이력         ← synced_at vs updated_at 자동 감지
├── 배출계수 적용 내역     ← applied_factor_id 스냅샷 목록
├── 데이터 품질 분포       ← M1/M2/E1/E2 비율 차트
├── 산정 버전 히스토리     ← v1/v2/v3 비교
├── 감사인 전용 뷰         ← 읽기 전용 공유 링크
└── 증빙 패키지 내보내기   ← PDF·Excel 5종 다운로드
```

## 4. 서브 메뉴 상세 설계

### 4.1 데이터 계보 추적

요건 체크리스트의 [↗] 클릭 또는 사이드 메뉴 진입 시 표시.

```

├── 고정연소: 852.4 tCO₂e
│   ├── 원천 탭: 연료·차량 탭 [바로가기↗]
│   ├── 원천 파일: ERP_FUEL_PURCHASE.csv  60건
│   ├── 동기화: 2024-02-05 07:30 (ERP DB Direct)
│   ├── 수동 수정: 1건 ⚠️  (user_002, 2024-02-10)
│   ├── 배출계수: EF-LNG-2023 (2.159 tCO₂e/TJ, 환경부, AR5)
│   └── 산식: consumption_amount × emission_factor × GWP
├── 이동연소: 298.6 tCO₂e
│   ├── 원천 파일: ERP_FUEL_PURCHASE.csv
```

## 5. 현재 화면 vs 목표 화면 비교

| 항목 | 현재 (❌) | 목표 (✅) |
|------|----------|----------|
| 첫 화면 | 내부통제 요약 대시보드 | IFRS S2 요건 체크리스트 |
| 공시기준 전환 | 없음 | 상단 탭으로 IFRS S2/GRI/KSSB/K-ETS/ESRS 전환 |
| 충족률 표시 | 증빙 구비율(내부 지표) | IFRS S2 요건 N/12 충족률 |
| 계보 드릴다운 | 없음 | 숫자 클릭 → 원천 레코드 추적 |
| 수동 조정 감지 | 없음 | synced_at vs updated_at 자동 감지 |
| 배출계수 버전 경고 | 없음 | MDG 현행값과 비교 자동 경고 |
| 더미데이터 연결 | 0건 (미연결) | 더미 CSV 기반 실제 수치 표시 |
| 감사인 공유 | 없음 | 읽기 전용 링크 생성 |
